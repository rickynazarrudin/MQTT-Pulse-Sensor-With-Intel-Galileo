1. subscribe.c : Program untuk mensubscribe data dari mosquitto broker dan untuk memasukkan data yang diperoleh ke localhost database.

2. publish.c : Program untuk mempublish data yang diperoleh dari xbee coordinator ke mosquitto broker.

3. poor-man'stelnet.ino : Sketch program arduino yang diupload ke board Intel Galileo untuk melihat IP dari board Intel Galileo.

4. pro.php : Program untuk menampilkan data dari localhost database dalam bentuk tabel.

5. real.php : Program untuk mengatur pro.php agar dapat menampilkan data secara real-time.

6. pulse.ino : Program yang diupload ke board arduino UNO untuk mendapatkan data dari sensor detak jantung.
